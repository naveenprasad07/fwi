Forest Fire Weather Index (FWI) Prediction

This web application uses a machine learning model to predict the Forest Fire Weather Index (FWI) based on weather-related features like temperature, humidity, wind speed, rainfall, and more. It helps in assessing fire risk levels in forested areas.

The model is trained on real-world weather and fire data, and it predicts the FWI based on the input values you provide. This is useful for early warnings and forest fire prevention strategies.

The Forest Fire Weather Index (FWI) is a numeric scale that indicates the potential risk and intensity of a wildfire based on current weather conditions. Higher values mean a greater chance and severity of fire.
